# Wiki Reader 示例知识库

这是一个用于测试 Wiki Reader 各项渲染功能的示例知识库。文件按主题组织，覆盖了 Markdown 的各种常见和进阶特性。

## 目录结构

```
wiki-example/
├── README.md                 ← 你在这里
├── getting-started.md        基础格式演示
├── code-examples.md          代码块与语法高亮
├── architecture.md           架构图与 Mermaid 流程图
├── img/                      图片资源目录
│   ├── landscape.jpg
│   ├── nature.jpg
│   ├── square.jpg
│   ├── thumbnail.jpg
│   └── banner.jpg
├── guide/
│   ├── images.md             图片引用与排版
│   ├── advanced.md           表格、脚注、数学公式
│   └── cross-reference.md    交叉引用与链接
└── notes/
    └── dev-log.md            开发日志（wiki 风格笔记）
```

## 快速导航

| 主题 | 文件 | 主要特性 |
|------|------|----------|
| 基础格式 | [getting-started.md](getting-started.md) | 标题、列表、引用、分隔线 |
| 代码高亮 | [code-examples.md](code-examples.md) | 多语言代码块、行内代码 |
| 架构图 | [architecture.md](architecture.md) | Mermaid 流程图、时序图 |
| 图片引用 | [guide/images.md](guide/images.md) | 行内图片、图片链接、尺寸 |
| 进阶排版 | [guide/advanced.md](guide/advanced.md) | 表格、脚注、数学公式 |
| 交叉引用 | [guide/cross-reference.md](guide/cross-reference.md) | 内部链接、锚点跳转 |
| 开发日志 | [notes/dev-log.md](notes/dev-log.md) | Wiki 风格的日常笔记 |

## 关于

本知识库中的图片来自 [Picsum Photos](https://picsum.photos/)（免费随机图片服务）。所有内容仅用于测试目的。
